# Widget Builder

Skill de creation de widgets embarquables pour ScaleHQ.

## Capacites

- **Widget chatbot** : creation d'un widget chatbot embarquable sur les sites des clients des agences SDR. Configuration du style (couleurs, position, avatar), du comportement (message d'accueil, delai d'apparition), et de la connexion au backend ScaleHQ.
- **Widget feedback** : creation d'un widget de collecte de feedback embarquable dans l'application ScaleHQ. Formulaire simplifie avec categorisation automatique (bug, feature, question), capture automatique du contexte (page, navigateur, session), et envoi vers le pipeline de feedback.
- **Widget KPI** : creation de widgets d'affichage de KPIs embarquables que les agences SDR peuvent integrer dans leurs propres dashboards ou portails clients. Configuration des metriques affichees (appels, RDV, taux de conversion), de la periode, et du style.
- **Configuration sans code** : interface de configuration visuelle pour personnaliser chaque widget — couleurs, textes, comportement, donnees affichees — sans ecrire de code.
- **Script d'integration** : generation d'un snippet JavaScript a copier-coller pour integrer le widget sur n'importe quel site. Le snippet est leger (< 50 KB) et n'impacte pas les performances du site hote.
- **Theming** : support des themes clair et sombre, avec detection automatique du theme du site hote et possibilite de forcer un theme specifique.
