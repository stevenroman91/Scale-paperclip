# Frontend QA

Skill de controle qualite automatise du frontend ScaleHQ.

## Capacites

- **Comparaison de screenshots** : capture de screenshots des pages et composants cles, comparaison pixel-par-pixel avec les references visuelles. Detection des regressions visuelles (changement de layout, disparition d'elements, decalages).
- **Audit d'accessibilite** : verification automatique de la conformite WCAG 2.1 AA — contrastes de couleur, presence des labels ARIA, navigation clavier fonctionnelle, roles semantiques corrects, textes alternatifs sur les images.
- **Scoring de performance Lighthouse** : execution des audits Lighthouse sur les pages principales — Performance, Accessibility, Best Practices, SEO. Alerte si un score descend sous le seuil (90 pour Performance, 95 pour Accessibility).
- **Tests responsive** : verification du rendu sur les breakpoints principaux (mobile 375px, tablette 768px, desktop 1280px, wide 1536px). Detection des debordements, des textes tronques, et des elements mal positionnes.
- **Tests de composants** : verification visuelle des composants du design system dans leurs differents etats (default, hover, focus, disabled, error, loading). Detection des inconsistances avec les specs du design system.
- **Rapport de regression** : generation d'un rapport de regression visuelle avec screenshots avant/apres, diff visuel, et liste des anomalies detectees.
